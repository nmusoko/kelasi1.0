Modifier l’écran de connexion de l’application de plateforme universitaire pour permettre à l’utilisateur de choisir son type de compte.

Ajouter une sélection du rôle utilisateur au-dessus du formulaire de connexion avec deux options :

Étudiant

Professeur

Le choix doit être présenté sous forme de boutons segmentés ou de cartes sélectionnables.

Lorsque l’utilisateur sélectionne :

Étudiant

le formulaire affiche : Identifiant étudiant / Email et mot de passe

Professeur

le formulaire affiche : Email professeur et mot de passe

Créer également deux flux de navigation différents après la connexion :

Étudiant →
Dashboard étudiant avec :

Mes cours

Matériels

Annonces

Devoirs

Chat

Calendrier

Profil

Professeur →
Dashboard professeur avec :

Mes cours

Gestion des matériels

Publication d’annonces

Création de devoirs

Liste des étudiants

Chat avec la classe

Le design doit rester moderne, minimaliste et cohérent avec une application EdTech.

Utiliser :

cartes modernes

composants arrondis

hiérarchie claire

navigation inférieure pour les étudiants

interface tableau de bord pour les professeurs

Style visuel : application universitaire moderne similaire à Canvas ou Notion, optimisée pour mobile et adaptée à une démonstration produit.
